L'offre est séduisante : un site internet professionnel, gratuit, avec maintenance incluse. Pour une PME au budget serré, c'est une aubaine. Mais en réalité, ce modèle économique est un piège.

## Comment une agence peut-elle offrir un site gratuitement

Elle ne le fait pas. Le site est gratuit à la livraison, mais il est construit sur une plateforme propriétaire. Cela signifie que vous ne possédez rien. Ni le code, ni le design, ni les données.

Le contrat qui accompagne ce site gratuit inclut un abonnement mensuel obligatoire : maintenance, hébergement, mises à jour de sécurité. Le tarif semble raisonnable au départ, mais il est indexé et peut grimper chaque année.

## Ce qui se passe quand vous voulez partir

Au bout de deux ou trois ans, si vous souhaitez changer de prestataire, vous découvrez la réalité :

- Votre site n'est pas exportable. Il ne fonctionne que sur la plateforme propriétaire.
- Vous devez repartir de zéro, avec un nouveau développement complet.
- Les textes et les images que vous avez fournis sont parfois revendiqués comme propriété de l'agence par des clauses cachées.

Vous n'aviez pas un site gratuit. Vous louiez un site sans savoir que le loyer allait durer éternellement.

## La seule question à poser avant de signer

Avant d'accepter une offre de site web, posez cette question à l'agence : « Suis-je propriétaire du code source et puis-je l'emporter chez un autre prestataire ? »

Si la réponse est non, ou si on vous explique que « ce n'est pas comme ça que ça fonctionne », fuyez. Un site internet est un actif d'entreprise. Vous devez en être le propriétaire exclusif.

## Comment reconnaître une offre saine

Un site professionnel est livré avec :

- Le code source complet, versionné sur un dépôt accessible
- Les identifiants d'administration, sans restriction
- Une documentation ou au minimum une passation claire

Le prix d'un site varie entre 3 000 et 8 000 CHF. C'est un investissement unique qui vous appartient, pas un loyer déguisé à vie.
