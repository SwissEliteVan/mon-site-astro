# L'ère de Tailwind CSS

Tailwind CSS a révolutionné la manière dont nous concevons les interfaces web et écrivons du CSS.

## Les avantages principaux

- **Rapidité de développement** : Ne quittez plus jamais votre fichier HTML, JSX ou Astro.
- **Maintenance facilitée** : Plus besoin de chercher où est définie une classe CSS obscure ou d'avoir peur de supprimer du code.
- **Design System natif** : Les espacements, les couleurs et la typographie sont toujours cohérents car ils s'appuient sur une échelle de valeurs partagée.

Voici un exemple de code :

```html
<button class="bg-indigo-600 hover:bg-indigo-500 text-white font-bold py-3 px-6 rounded-xl transition-colors">
  Bouton Tailwind
</button>
```

N'est-ce pas beaucoup plus simple de comprendre l'intention du bouton au premier coup d'œil ?
