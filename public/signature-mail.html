Copie effectuée — collez directement dans votre client mail.

Comment intégrer la signature dans votre client mail :
  1. Gmail : Cliquez "Copier (Gmail)" → Paramètres → Signature → Coller dans la zone de texte.
  2. Outlook : Cliquez "Copier le HTML" → Fichier → Options → Courrier → Signatures → Nouveau → coller dans l'éditeur HTML.
  3. Apple Mail : Cliquez "Copier (Gmail)" → Mail → Préférences → Signatures → coller dans la zone.
